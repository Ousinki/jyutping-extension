# 粵語懸浮詞典 (Cantonese Popup Dictionary)

一個 Chrome 擴展，滑鼠懸停即可顯示中文字詞的粵語發音（粵拼/Yale）和英文釋義。

![Demo Screenshot](docs/screenshot.png)

## ✨ 功能特點

- **滑鼠懸停即時查詞**：將滑鼠移到任何中文字上，即刻顯示粵語發音
- **雙拼音系統**：支援粵拼 (Jyutping) 和 Yale 兩種標註方式
- **英文解釋**：提供詳細英文翻譯和釋義
- **超過 23 萬詞條**：涵蓋常用詞彙、成語、俚語
- **Shadow DOM 支持**：兼容使用 Web Components 的現代網站（如 Bilibili）

## 📦 安裝

### Chrome Web Store（推薦）
*審核中...*

### 手動安裝
1. 下載或 Clone 本倉庫
2. 打開 Chrome，進入 `chrome://extensions/`
3. 開啟右上角的「開發者模式」
4. 點擊「載入未封裝項目」
5. 選擇項目資料夾

## 🔧 使用方法

1. 安裝後自動在所有網頁生效
2. 將滑鼠移到任何中文字上
3. 彈窗會顯示：
   - 繁體/簡體字
   - 粵拼讀音
   - 英文解釋

## ⚙️ 設定

點擊擴展圖標可以：
- 開啟/關閉擴展
- 切換粵拼/Yale 顯示模式
